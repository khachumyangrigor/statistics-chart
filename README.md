# Statistics Chart

Interactive Line Chart для визуализации результатов A/B тестирования с отображением conversion rate.

## Chosen Visualization Library

**Recharts** - библиотека для построения графиков на основе React и D3. Выбрана за простоту использования, хорошую документацию и поддержку интерактивных функций.

## Implemented and Bonus Features

### Основные требования:
- ✅ Отображение conversion rate в виде line chart для всех вариаций
- ✅ Все значения отображаются в процентах
- ✅ При наведении показывается вертикальная линия и попап с данными за день
- ✅ Минимум одна вариация всегда выбрана (нельзя отключить последнюю)
- ✅ При переключении вариаций оси X и Y автоматически адаптируются к видимым данным
- ✅ Адаптивный дизайн для экранов от 671px до 1300px
- ✅ Селектор вариаций (чекбоксы для выбора отображаемых линий)
- ✅ Переключатель Day / Week (дневной и недельный режим)

### Бонусные функции:
- ✅ **Zoom** - возможность масштабирования графика через Brush компонент
- ✅ **Line style selector** - выбор стиля линий: Line (линейный), Smooth (сглаженный), Area (заливка)
- ✅ **Light / Dark theme toggle** - переключение между светлой и темной темой
- ✅ **Export chart to PNG** - экспорт графика в PNG формат

## Local Setup Instructions

### Предварительные требования:
- Node.js (версия 18 или выше)
- npm или yarn

### Установка и запуск:

1. Клонируйте репозиторий:
```bash
git clone https://github.com/[YOUR_USERNAME]/statistics-chart.git
cd statistics-chart
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите dev-сервер:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

4. Сборка для production:
```bash
npm run build
```

5. Просмотр production сборки:
```bash
npm run preview
```
